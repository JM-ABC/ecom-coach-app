Background

기획 의도

정보의 파편화

플랫폼별 일정과 날씨, 뉴스, 트렌드 데이터가 흩어져 있어 MD의 기획 리소스가 단순 데이터 수집에 낭비되는 문제를 해결하고자 했습니다.

기획의 병목 현상

시장 변화에 따른 프로모션 기획서를 수동으로 작성하는 방식의 한계를 극복하고 대응 속도를 높이는 데 집중했습니다.

심리스한 연결

데이터 분석부터 기획안 도출까지 하나의 흐름으로 이어지는 효율적인 업무 환경 구축을 목표로 했습니다.

"MD의 업무 효율을 극대화하는 도구"
업무 프로세스 자동화

Key Features

주요 기능

📅

통합 마케팅 캘린더

주요 플랫폼 이벤트를 한눈에 파악하고 전략적으로 배분할 수 있는 통합 뷰를 제공합니다.

  • 카테고리별 스마트 필터링
  • 커스텀 이벤트 관리
🛰️

외부 데이터 연동

기상청 API 및 네이버 쇼핑 트렌드를 실시간 연동하여 기획에 필요한 객관적 근거를 제공합니다.

  • 날씨 마케팅 소스 제공
  • 실시간 뉴스 브리핑
🪄

AI 기획서 생성

Gemini 모델을 통해 선택한 이벤트에 최적화된 기획 초안을 즉시 생성하여 업무 속도를 높입니다.

  • 맞춤형 프로모션 전략 제안
  • 실시간 텍스트 생성 기술
