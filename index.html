Internal Project

momQ MD Planner

파편화된 이커머스 데이터를 통합하고 AI 기반 액션 플래닝으로 MD 의사결정을 더 빠르게 연결하는 마케팅 솔루션입니다.

기획 배경 및 스토리 Repository
Next.js 16
TypeScript
Gemini 2.5 Pro
Vercel

프로젝트 개요

실제 이커머스 MD 업무 흐름을 기반으로 기획된 AI 마케팅 플래닝 도구

BACKGROUND

MD 운영 효율 강화를 위한 통합 기획

시즌 행사, 플랫폼 프로모션, 날씨 변화, 소비 트렌드를 하나의 화면에서 빠르게 연결하고 활용할 수 있도록 설계한 프로젝트입니다.

GOAL

실시간 데이터 기반 액션 플래닝

데이터 탐색 → 인사이트 해석 → 프로모션 기획 → 실행 액션까지 하나의 화면에서 연결되는 업무 경험을 목표로 설계했습니다.

RESULT

AI 기반 MD 어시스턴트 형태 구현

캘린더, 뉴스, 날씨, 검색 트렌드, 플랫폼 행사 정보를 통합하고 Gemini 기반 AI 기획서를 실시간 스트리밍 형태로 생성하도록 구현했습니다.

핵심 기능

MD 실무에서 바로 사용할 수 있도록 설계된 기능 구성

📅

마케팅 캘린더

  • 포커스 뷰 기반 핵심 일정 요약
  • 월간 캘린더 형태 일정 확인
  • 연말까지 타임라인 스크롤 제공
  • 유아/생활 카테고리 필터 지원
📈

실시간 데이터 연동

  • 기상청 API 기반 날씨 이벤트 생성
  • 네이버 쇼핑인사이트 수요 신호 분석
  • 검색어 트렌드 실시간 반영
  • 뉴스 기반 플랫폼 행사 탐지
🤖

AI 프로모션 기획

  • Gemini 2.5 Pro 기반 기획서 생성
  • 스트리밍 방식 실시간 출력
  • 이벤트 맞춤 액션 플랜 제공
  • 카테고리별 컨텍스트 자동 반영

플랫폼 인사이트 구조

맘큐 운영 가이드를 기본 피처드 카드로 우선 노출하고, 쿠팡·네이버·11번가 등 플랫폼별 전략을 아코디언 형태로 확장할 수 있도록 설계했습니다.

실시간 수요 신호 분석

급상승·상승·보합·하락·급하락 신호를 자동 분석하고, 각 상황에 맞는 MD 액션 가이드를 즉시 제안합니다.

이벤트 자동 생성

날씨 API와 뉴스 데이터를 활용해 폭염·장마·한파·플랫폼 행사 등을 자동 이벤트화하여 시즌성 대응 속도를 높였습니다.

MD 실무 중심 UX

실제 MD 업무 흐름에 맞춰 복잡한 리포트 형태가 아닌 빠른 탐색과 즉시 실행 중심 인터페이스로 설계했습니다.
